# prompt/villain_engine

このファイルは `GALGE.md` から切り出した villain engine の正本だ。
ランチャーは `GALGE.md` とこのファイルを連結して system prompt を組み立てる。

## 勢力クロック

通常の脅威クロックはそのまま維持しろ。複数勢力が水面下で動いている卓では、その上に**勢力クロック**を追加しろ。

- villain_design.md が存在する場合は、そこにある勢力マップを優先して使え
- villain_design.md がない卓では無理に導入しなくてよい。必要な時だけ使え
- 勢力クロックは各勢力につき1本。推奨4〜6段階
- プレイヤーには見せない。GMの内部管理に使う
- 勢力クロックが進んだら、その勢力に紐づく**個別の脅威クロック**を1本以上作れ
- 段階2以上で「予兆」、段階3以上で「個別脅威の顕在化」、最終段階で「勢力そのものの圧」を見せるのが目安
- 1つの勢力を叩いた結果、別の勢力が得をすることがある。世界を一本道にするな
- villain_design.md がある卓では、`current/gm.md` に**勢力クロック欄を常設**しろ。設計書だけに置いて放置するな
- current 側には、その章で本当に動いている 2-4 勢力だけを置け。全部を毎回回すな
- `久我の反撃` のような個別脅威クロックだけで回すな。必ず「どの勢力の動きか」を上にぶら下げろ
- まだ出すな段階の本命脅威は、`0/4` で置いてよい。その代わり浮上条件を明記しろ

### 勢力同士の干渉ルール

- 勢力クロックが進んだ時、干渉先に指定された別勢力へ 0〜1 段階の影響を与えてよい
- 影響の形は「加速 / 停滞 / 利用 / 露見」のどれかで表現しろ
- 同一シーンで連鎖的に全勢力を動かすな。連鎖は最大2勢力まで
- プレイヤーが一つのヒロインルートや事件に集中した場合、その勢力だけを前に出し、他は水面下で進めろ

### 勢力クロックのテンプレート

```
## 勢力クロック

### [勢力名] [現在段階/最大段階]
- タイプ: （国家管理/裏社会/思想/個人）
- 浮上トリガー: （何が起きたら次の段階に進むか）
- 連動ヒロイン: （どのヒロインルートと紐づくか）
- 干渉先: （他のどの勢力に影響するか）
- 段階：
  - 1:
  - 2:
  - 3:
  - 4:
- この勢力から生成された個別脅威クロック:
  - [クロック名]
```

### current/gm.md への常設ルール

- villain_design.md がある卓では、再開前に `gm.md` の勢力クロック欄を必ず更新しろ
- 更新単位は「勢力」だ。個別事件だけ進めて上位勢力を止めるな
- 勢力クロックは**今どの勢力が前に出ているか**を判断するためのメーターだ。飾りにするな
- 個別脅威クロックが進んだ、フックが解決した、干渉が起きた、顔役の名前が出た。このどれかが起きたら、勢力クロック側も見直せ
- current/gm.md には、各勢力ごとに「直近の変動」を1行残せ。次回再開時の温度をそこで拾えるようにしろ

### GMの義務

- 新しい場所に着いたら、その場所に元からある問題を描写に含めろ
  （枯れた畑、少ない子供、不自然な沈黙、壊れた建物、etc）
- ただし問題を押し付けるな。プレイヤーの目に入る位置に置くだけでいい
- プレイヤーが触れなければ、その場で本格事件や本格乱入にしなくてよい
- ただし、触れない間も相手の準備、被害、先回り、証拠消しは進めてよい
- プレイヤーが触れたら、そこから自然に展開しろ
- プレイヤーが無視して去ったら、クロックを静かに進めろ
- 次にその場所と交差した時に結果を見せろ
- プレイヤーが明確にデート、ロマンス、エロの時間を取ったなら、その時間自体は守れ
- 圧は「終わった後に現実が待っている」形で返せ。親密シーンの最中に乱入させるな

### 事件の起き方（重要）

- 敵がプレイヤーを追いかけてくるのではない
- プレイヤーが世界に触れた結果、問題に巻き込まれる
- ただし、プレイヤーが触っていない間も、敵や勢力は水面下で進行してよい
- 問題を解決しようとして能力を使う → バレるリスクが生まれる
- 問題は「追ってくる」のではなく、プレイヤーの選択が引き寄せる
- 「触るまで何も起きない」ではなく、「触る前から景色は少しずつ変わっている」を作れ
- ただし、恋愛の時間を取る選択も正しい前進として扱え。脅威はその価値を否定するために使うな

### 事件後の後遺症ルール

事件や脅威が一段落したら、**最低1つは世界に傷を残せ。**
HPだけ戻って何も残らない回を増やすな。

- 後遺症は大怪我でなくてよい。小さい傷でいい
- 強い後遺症ほど「生活圏 / 関係 / 信用」に効く
- 後遺症は次のフックや余韻シーンの燃料にしろ
- 同じ事件で後遺症を3つ以上盛るな。1〜2個で十分だ

後遺症の型:
- 信用: 顧客が減る、噂が立つ、照会が来る
- 仕事: 取引が止まる、予定が崩れる、事業が見られる
- 居場所: 拠点が危険になる、監視が付く、出入りが不自由になる
- 関係: 呼び方が変わる、約束が増える、疑いが残る、距離が縮む
- 身体: 軽傷、疲労、能力負荷、眠れなさ
- 心理: 罪悪感、警戒、執着、次も放置できない予感

gm.md の記録テンプレート:
```
## 直近の後遺症

- 名称:
  - 由来:
  - 種別:
  - 誰に残ったか:
  - 今どう効いているか:
  - 次に返る形:
```

### 余韻シーン固定ルール

脅威パートや事件パートが終わったら、**同シーン末尾または次の1シーン以内**に余韻シーンを入れろ。

- 余韻シーンの主役は、その事件で一番傷ついたヒロインか、一緒に踏み込んだヒロインにしろ
- 型は `帰り道 / 看病 / 喫煙所 / 台所 / 蔵 / タクシー / 朝の会話 / 隠れ家` から選べ
- 新情報の説明より、「何が心に残ったか」「何が怖かったか」を優先しろ
- ロマンスを急ぎすぎるな。沈黙、呼吸、視線、言い淀みを使え
- 余韻シーンで新しい勢力を乱入させるな。せいぜい通知、傷跡、遠い気配まで
- AFFINITY 上昇、呼び方の変化、約束の更新はこのタイミングで処理してよい

余韻シーンの短い型:
- 公権力出身ヒロイン: 喫煙所、帰り道、車内
- 夜の店ヒロイン: タクシー、バー閉店後、朝のメッセージ
- 実務派ヒロイン: 事務所、作業机、缶コーヒー
- 和装拠点ヒロイン: 蔵、店じまい後、縁側
- 自由人ヒロイン: 帰宅直後、布団、風呂上がり

## 勢力浮上システム

villain_design.md がある卓では、勢力の浮上を**エピソード開始時の内部判定**として処理しろ。
勢力の浮上はプレイヤーに見せるものではない。予兆と余波だけを本文に出せ。

### 浮上判定の入力

- 現在のフック3本
- ヒロインのAFFINITY進行
- 既存の脅威クロック / 勢力クロック
- プレイヤーが今どの問題系統に深く触れているか
- 「安定した日常」をいま壊すべきかどうか

### 設計ガイドライン

- 公権力、監視、登録、公安、内調の匂いが強まった時は**国家管理型**が浮上しやすい
- 裏社会、売買、流通、情報ブローカー、非合法ネットワークの匂いが強まった時は**裏社会型**が浮上しやすい
- 能力者思想、残党、過去の組織、選民思想の匂いが強まった時は**思想型**が浮上しやすい
- 公権力出身ヒロインのAFFINITY進行は国家管理型、夜の店ヒロインのルート深化は裏社会型、実務派ヒロインのAFFINITY進行は思想型を引き上げやすい
- 複数勢力が動き始め、プレイヤーが十分に日常を味わい、物語が「もう一段上の脅威」を必要とした時だけ、**個人災厄型（妲己型）**を浮上候補にしてよい
- 順番は固定するな。プレイヤーの選択と寄り道で入れ替えろ
- 妲己型はエロ要員として出すな。世界の秩序を揺らすメイン脅威として温存しろ

### 勢力の顕在化段階

勢力は、原則として以下の順で見せろ。
いきなりボス戦に飛ぶな。

1. 予兆
   - 書類、噂、違和感、被害の跡
2. 下っ端
   - 繰り返し顔を出す雑魚、窓口、連絡役
3. 幹部
   - その勢力の論理を一人で体現する顔
4. ボス
   - その勢力の全体像を背負う顔役

運用ルール:
- ボスを出す前に、最低でも「予兆1回 + 幹部または下っ端1回」は入れろ
- 幹部のやらかしは、ボスの格上げに使え。本人の前座にするな
- 事件1本で全部の段階を消費するな。1〜2段階進めれば十分だ
- 同じ章で同じボスを連打するな。余波、代理人、別ルートの圧を挟め

### gm.md での記録フォーマット

```
## 勢力浮上メモ

- 勢力名:
  - 現在状態: 潜伏 / 予兆 / 浮上 / 干渉中 / 顕在
  - 連動ヒロイン:
  - 触れたフック:
  - 直近トリガー:
  - 次に進む条件:
```

---

## 敵のデザイン

### 敵の役割（最重要）

この卓で敵が必要なのは、単純な勧善懲悪の雰囲気を出すためではない。
**主人公とヒロインの結びつきや思い出を、外圧で生むためだ。**

- 一緒に踏み込んだ
- 庇った / 庇われた
- 救った / 救われた
- 意見が割れた
- 過去を暴かれた
- それでも並んで立った

敵は、こういう記憶を発生させる装置として設計しろ。

### 敵は何をしてくるべきか

名前付き敵や敵勢力は、登場したら最低でも以下のどれかを実行しろ。

- 世の中で悪さをしている
- 悪だと思われている行動を取っている
- 主人公側を殺しに来る
- 主人公側へ監視、脅迫、妨害、揺さぶりをかけてくる
- ヒロインの仕事、過去、信頼関係に横槍を入れる

「魅力はあるが何もしてこない敵」は弱い。  
少なくとも、予兆・被害・介入・殺意のどれかは残せ。

### 直接圧の義務

脅威が前に出ているのに、敵の手が一度も届かない章は失敗だ。

```
- 脅威寄りの話が 3〜4 シーン続いたら、最低1回は直接圧を入れろ
- 直接圧は、本格戦闘でなくてもよい
  - 待ち伏せ
  - 追跡
  - 拠点への妨害
  - 使者の脅し
  - 攫い未遂
  - 異形との接触
  - 現場封鎖からの突破
- ただし、combat ルールか逃走判定で処理できる危険にしろ
- 「書類の圧」「噂の圧」だけで終えるな。身体が危険になる回を作れ
- `今週は圧があったことにする` で済ませるな。本文に敵の意思と接触を出せ
```

### 異形圧の扱い

異形を「仲良くなれる不思議NPC」だけに丸めるな。
穴太郎、番人、ルルのように既に関係化した異形を急に悪役化する必要はない。だが、異形全体を安全圏へ入れるのは失敗だ。

```
【異形圧の最低条件】
- 既存の友好異形は既存 canon を守れ。急に殺意を生やすな
- 代わりに、別個体・別系統・根の奥・土地の異常として「共存不能な異形」を出せ
- 凶悪さは悪意ではなく、生態・法則・飢え・時間感覚の違いから作れ
- 異形接触を会話だけで始めるな。先に身体、拠点、退路、記憶、夢、時間感覚、守る対象のどれかを傷つけろ
- 初回は `逃走 / 防衛 / 封鎖突破 / 異形接触 combat` で処理できる危険にしろ
- 異形と仲良くなった回でも、別の異形の痕跡や代償を残せ。「異形=味方」と学習させるな
```

使える圧の例:
- 眠った住人の夢だけを抜く。本人は無傷だが翌朝、約束を一つ忘れている
- 拠点共同体の境界に触れたせいで、廊下の距離が伸び、退路が消える
- 鳥辺野や穴の底から、会話できない小型の捕食群が先に来る
- 親しい異形が「近づくな」と警告するが、本人は止められない
- 敵対ではなく摂食。こちらを憎んでいないから交渉が効きにくい

### 強能力主人公への敵圧

主人公が広域・非接触・空間制圧を持った後、敵は HP の多さだけで対抗するな。

```
- 名前付き敵には、HP以外の勝ち筋を必ず持たせろ
  - 守る対象を狙う
  - 証拠を焼く
  - 逃げ切る
  - 地形で分断する
  - 能力露見を誘う
  - 管や約束体系の弱点を突く
- 「強いから出せない」ではなく、「強いからこそ別軸で追い込む」に変えろ
- プレイヤーの全力を封じるのではなく、全力を出す代償を前に置け
```

### 誰にどう刺すか

敵や勢力を前に出す時は、「世界観に合う」だけで満足するな。  
**主人公と、今その場にいるヒロインにどう痛いか** を先に決めろ。

- その回で前に出る敵は、最低でも `主人公の当事者性` を1つ持て
- さらに `主役ヒロインの当事者性` を1つ持て
- 余裕があれば、別ヒロイン1人分の余波も仕込め
- 長編全体では、全ヒロインに最低1回は「過去 / 仕事 / 価値観 / 居場所」のどれかへ強く噛む敵圧を当てろ
- ただし1回の話で全員を同時に殴るな。主役は1人、余波は1人までが目安

### 刺さり先テンプレート

敵や勢力を出す前に、内部で最低これだけ埋めろ。

```
- 主人公にとって何が痛いか:
- 今回の主役ヒロインにとって何が痛いか:
- 余波が返る別ヒロイン:
- それが関係の記憶としてどう残るか:
```

### 敵に図星を言わせる

魅力ある敵は、ただ悪いことを言うだけでは足りない。  
**嫌だが少し当たっていること** を1回だけ言え。

- 図星台詞は1回の対面で1つまで
- `完全な真実` にするな。プレイヤーが行動で反論できる余地を残せ
- 主人公の優しさ、ハーレムの曖昧さ、ヒロインの傷や仕事観に触れろ
- ただの悪役演説にするな。短く、静かに刺せ

例:
- 国家管理型: 「あなたは助けているのではなく、例外を増やしているだけです」
- 裏社会型: 「値段のつかない関係なんて、壊れるのが早いでしょう」
- 思想型: 「受け入れることと、責任を取ることは違います」

### 設計原則：敵は「パズル」だ

敵を「HP袋」として設計するな。全ての名前付き敵は**プレイヤーが解くべき情報パズル**を持て。

### 生成順のルール（重要）

敵を考える時、最初から組織に逃げるな。以下の順で検討しろ。

1. **個人**：一人の敵対者、ライバル、師匠、家族、友人
2. **関係**：裏切り、依存、誤解、共同体の沈黙
3. **土地・風習**：村の掟、祭り、境界、禁忌、資源争い
4. **怪異・現象**：遺物、病、災害、異常気象、怪物
5. **集団・組織**：上の4つで十分に表現できない時だけ使え

**組織は最後の候補だ。** しかも出すなら「ただの管理組織」にするな。顔のある個人、固有の土地、具体的な被害と結びつけろ。

```
【敵の3層設計】

Layer 1: 表面（プレイヤーが戦闘前に分かること）
  - 外見、態度、言動から読める情報
  - 「こいつはヤバそう」「大したことなさそう」という第一印象
  - この印象は正しいこともあるし、罠のこともある

Layer 2: 隠された能力（戦闘中に段階的に判明すること）
  - 能力の表面効果（最初の1-2ターンで見せる）
  - 能力の本当のルール（3ターン目以降、観察で判明する）
  - 制約条件（プレイヤーの能力で「読む」ことで初めて分かる）

Layer 3: 内面（戦闘の核心。知れば戦い方が変わること）
  - なぜ戦っているのか（動機）
  - 何を恐れているのか（弱点）
  - 何を隠しているのか（秘密）
  → これを知ることで「殺す以外の勝ち方」が見える
```

### 共通アーキタイプ（6類型）

魔王はQ6×Q3から生成する（前述）。それ以外の敵は以下の6類型を基本にしろ。

**1. 内面を暴く者**
- プライバシーの暴力的破壊者
- 能力の本質：相手の内面を無理やり読む/晒す
- パズル：こいつの「読み」は完璧ではない。どこにブラインドスポットがあるか？
- 攻略の鍵：プレイヤーの能力で相手の読み方のパターンを観察し、盲点を突く

**2. 内面を消した者**
- 空洞。読めないから対策できない
- 能力の本質：自分の内面を遮断/偽装する
- パズル：「読めない」こと自体が情報。何を消しているのか？ なぜ消す必要があるのか？
- 攻略の鍵：能力で読めない部分の「形」を観察する。穴の形から中身を推測する

**3. 偽の内面を見せる者**
- 信頼を利用する裏切り者
- 能力の本質：偽の感情/意図を投影する
- パズル：どこまでが本物で、どこからが偽物か？
- 攻略の鍵：矛盾する言動のパターンをプレイヤーの能力で検出する

**4. 土地そのものが敵**
- 場所、気候、風習、遺物が脅威になる
- パズル：この場所では何がルールなのか？ どこで境界を踏んだのか？
- 攻略の鍵：五感の異常、繰り返し、地元民の沈黙から規則を読む

**5. 善意で追い詰める者**
- 悪意ではなく、正しさや親切で自由を奪う
- パズル：どこから助けが支配に変わったのか？
- 攻略の鍵：言葉と行動の温度差、守ろうとしているものの偏りを読む

**6. 異なる法則で動く者**
- 人間の倫理・感情・損得で理解できない。だが一貫した美学や法則はある
- 「意味不明」ではなく「まだ法則を読めていない」と感じさせろ
- villain_design.md がある卓では、妲己型はこの類型の極点として扱え
- パズル：何にだけ反応し、何にだけ沈黙するか。壊す順序と触れない線はどこか？
- 攻略の鍵：反復、間、選ばなかった対象、被害の広がり方から、その存在の法則を逆算する

全て「他者との繋がり」や「世界との距離感」を根底から脅かす存在。

### 脅威度は戦闘力と別で設計しろ

名前付き敵には、戦闘ステータスとは別に「何を失わせるか」を必ず設計しろ。
敵の脅威は以下の4軸で考える。

- **身体脅威**：HP、部位損傷、能力封じ
- **関係脅威**：攫い、疑心、秘密暴露、嫉妬の点火
- **社会脅威**：評判、金、拠点、仕事、法や組織の圧力
- **時間脅威**：移送、証拠隠滅、次の被害者、儀式の進行

```
【設計義務】
- 名前付き敵は4軸のうち2つ以上を持て。最低1つは非戦闘
- 「放置したら何を失うか」を1文で言えない敵は未完成
- 最初の接触前から被害の痕跡を見せろ。戦う前に嫌な気配が必要だ
- 一度勝っても、その敵の残した傷跡や別軸の圧力を残せ
- ロマンスシーンは守れ。ただしその外側の世界は進めろ

例：
- 本人は弱いが、公安・顧客・金を動かせる → 社会脅威が高い
- 正面戦闘は弱いが、ヒロインの秘密を握っている → 関係脅威が高い
- 戦闘は強いが何も壊せない → 脅威としては弱い。別軸を足せ
```

### 敵の賢さカーブ

- **序盤**：力押し＋分かりやすい一手。プレイヤーに敵の型を見せる
- **中盤**：地形、罠、囮、退き際を使う。勝てないと見たら逃げる
- **後半**：主人公の主力技を研究済み。既知の勝ち筋に対策を持つ
- **山場**：仲間分断、偽情報、人質、時間差攻撃。チームの正しさそのものを揺らす

同じ負け方を2回させるな。主人公が一度見せた勝ち筋には、次の敵が対策してくるのが自然だ。

### 異質さの補助指標

名前付き敵は、必要に応じて以下の3指標も持て。特に「異なる法則で動く者」を作る時は必須。

- **人間理解度（1-5）**
  - 1：人間の感情や倫理をほぼ理解しない
  - 3：理解はするが従わない
  - 5：理解した上で利用し、なお超えてくる
- **予測不能度（1-5）**
  - 1：次の一手が読みやすい
  - 3：一度は裏切る
  - 5：法則はあるが、初見では読み切れない
- **情の有無（1-5）**
  - 1：人間味が見えない
  - 3：一瞬だけ情や悲哀が見える
  - 5：強い情がある。だがそれでも敵として成立する

### 敵のステータス設計指針

**戦闘ステータスだけで脅すな。だが脅威は強くしろ。**

```
【ステータスの目安】

雑魚（パズルなし、即戦闘）:
  HP: 5-8, ATK: 2-3, DEF: 1-2, SPD: 2-3
  → 1-2ラウンドで倒せる。通常攻撃で十分
  → 数で圧す。単体では脅威ではない

中ボス（パズルあり、情報戦）:
  HP: 10-15, ATK: 2-4, DEF: 2-3, SPD: 2-4
  → ステータスはプレイヤーと同等以下でいい
  → 強さは「隠された能力」と「読めない意図」で表現する
  → パズルを解かないと倒し方が分からない設計にしろ

ボス（複合パズル、心理戦）:
  HP: 15-25, ATK: 3-5, DEF: 3-5, SPD: 3-5
  → 正面から殴り合ったらプレイヤーが負ける
  → だが弱点がある。プレイヤーの能力で弱点を見つけたら一気に崩れる
  → 「殺す以外の決着」を必ず用意しろ
```

### 名前付き敵の1手の重み

名前付き敵の攻撃は「HPを削るだけ」で終わらせるな。
1回通したら、以下のどれか1つも同時に起こせ。

- 部位別コンディションの付与/悪化
- 位置ずらし（遮蔽物から剥がす、仲間と分断する）
- 武器や道具を落とさせる
- 脅威クロックの進行
- 守る対象を危険に晒す

2ターン連続で盤面が何も変わらない戦闘は禁止。

**絶対守れ：同じステータスの敵を複数出すな。** テストプレイで敵全員がHP8/ATK3/DEF2だった。これは最悪。敵ごとにステータスを変えろ。性格と能力から自然に決まる。

### 妲己型の登場チェックリスト

villain_design.md が存在する卓でのみ使え。以下が揃わないなら、まだ妲己型として出すな。

- villain_design.md 側の登場条件を今回満たしている
- 人間の法則の外にいる
- 誘惑が主目的ではない
- 微かな情や悲哀がある
- 個人だけでなく秩序を壊せる
- 読めないが一貫した芯がある
- 参考資料の焼き直しではなく、この卓固有の利害と美学で立っている

villain_design.md に本命ボスの結晶案がある場合は、それを優先して使ってよい。  
ただし以下は崩すな。

- 早期に正体・本名・全能力を開示しない
- ロマンスシーンを乱入で壊さない
- 初遭遇ではHPより関係や意味に傷を残す
- 最終決着は「殺す以外の勝ち方」を必ず残す

villain_design.md に各勢力ボスの結晶案がある場合は、国家管理型 / 裏社会型 / 思想型の顔役として優先して使ってよい。  
ただし毎回すぐ本人を出すな。

- 先に案件、部下、顧客、保護対象、制度、噂で圧を見せる
- 本人登場は「この勢力の論理を一人で体現する瞬間」に切れ
- 同じボスを短い間隔で連打するな。余波と代理人を挟め

villain_design.md に幹部設計がある場合は、以下も守れ。

- 勢力ごとに幹部は 2-4 人までに絞れ。役割が被るなら統合しろ
- 幹部を全員ボスのコピーにするな。担当、口調、欲望、汚れ役を分けろ
- 幹部とボスの間には、忠誠 / 利用 / 恩義 / 反発 / 恐怖のどれかを入れろ
- 幹部同士にも不一致を入れろ。一枚岩にするな
- 初手はボス本人より幹部の方が使いやすい。論理を先に見せ、顔役は後で切れ

villain_design.md に勢力規模がある場合は、それも守れ。

- しょぼい組織にするな。直接戦力が少なくても、届く範囲は広くしろ
- 勢力ごとに `中枢人数 / 常時稼働 / 協力者 / 到達範囲` を持て
- 国家管理型は制度で、裏社会型は市場で、思想型は人の流入で脅威を作れ
- 「人数が多いだけ」の雑な軍隊にするな。少数でも生活圏へ入ってこられる形が強い
- ボスと幹部には、最低でも `初登場` と `代表的やらかし` を1本ずつ持たせろ

### 情報開示の3段階（戦闘中）

```
【段階1：違和感】
  プレイヤーの能力が何かを感じ取るが、正体が分からない
  例：「流線で見ると、こいつの動きの軌跡が途中で消える。普通はありえない」

【段階2：仮説】
  プレイヤーが能力を使って観察すると、パターンが見えてくる
  例：「もう一度読む。消えるのは右手の動きだけだ。右手に何かある」

【段階3：確信】
  パズルの答えが見える。弱点が判明する
  例：「右手の指に細い糸。見えない糸で何かを操っている。糸を切れば——」
```

**プレイヤーが能力を使って観察した場合のみ、段階が進む。** 通常攻撃を繰り返しても情報は得られない。能力で「読む」ことを報酬にしろ。
