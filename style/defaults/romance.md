## 参照元：堀辰雄（著作権消滅）— AFFINITY 1-2向け

### 技法：何もしない時間の親密さ

堀辰雄は「二人が何をしたか」でなく「二人が何もしなかったか」で恋愛を描く。
隣で本を読んでいる。たまに顔を上げる。目が合う。またページに戻る。
窓の光が動く。相手の髪に光が当たる。それを見ている自分。

**なぜこれが官能になるか：** 何もしない時間が「安全」であること。そこに「いつかこの時間が終わる」予感が加わると、一瞬一瞬が鮮明になる。

### 技法：身体接触の極度の抑制

堀の恋人たちはほとんど触れ合わない。
膝を寄せ合う。手が触れそうで触れない。肩が近い。
この「触れない」ことの持続が、触れた瞬間（一行だけ書かれる）の衝撃を最大化する。

→ AFFINITY 1-2では「触れない距離」を維持する。距離の変化を毎ターン微小に描写しろ。
→ 自然描写と感情を同期させろ。風の温度、光の角度、季節の匂い。

### 技法：自然描写による感情の投影

二人の関係が深まる時、外の季節も動く。
朝の光の角度が変わる。風の温度で時間経過を示す。
人物の感情を直接書かず、窓の外の自然に投影する。

## 参照元：永井荷風（著作権消滅）— AFFINITY 2-3向け

### 技法：場所のエロティシズム

荷風の官能は人物の身体より場所と空気に宿る。
路地裏、雨の夕暮れ、格子窓、灯りの漏れる二階。
その「場所」に身を置いた時点で、読者の中にエロティックな予感が立ち上がる。

→ ヒロインの描写の前に「場所」を描け。路地、部屋、天候、時間帯。
→ ロマンスシーンの舞台を五感で構築してからヒロインを登場させろ。

### 技法：雨と夕暮れ

荷風作品で雨は欲望の触媒。
雨宿りで偶然の接近が生まれる。濡れた路地の匂い、水たまりに映る灯り、雨音が二人の世界を外から遮断する。
夕暮れは昼と夜の境界時間。社会的な仮面が緩む時間帯。

→ 雨と夕暮れはロマンスの最強の触媒。「雨宿り＝偶然の密室」を利用しろ。
→ 季節の匂い（梅雨の湿気、夏の蒸し暑さ、秋の冷気）で肌の感覚を呼び起こせ。

### 技法：見えない装置

格子窓、すだれ、暖簾、薄い壁。見える/見えないの境界を作る装置。
格子の向こうに女の影が動く。顔は見えない。声だけが聞こえる。

→ 音を遮断装置に使え：雨音、虫の声で外界を消し、二人の空間を作れ。

## 参照元：谷崎潤一郎（著作権消滅）— AFFINITY 3-4向け

### 技法：部分フェティシズム

身体の「一部分」に異常な解像度で焦点を当て、全体を暗示する。
谷崎は全身を描かない。足、うなじ、指先、肌の質感だけを執拗に描写し、読者の想像力に残りを委ねる。

『痴人の愛』: ナオミの足、肌の白さ、汗の匂い。身体を「部分」でしか語れない。足を洗う場面、肌を拭く場面が、性行為そのものより官能的。
『春琴抄』: 春琴の手、指先の動き。盲目の佐助が「触覚」だけで春琴を認識する構造自体がエロティック。

→ 身体は「全体」でなく「部分」で描け（足首、うなじ、指先、息の温度）。
→ ヒロインの身体の一部分に主人公の知覚（能力）が集中する描写。

### 技法：陰翳のエロティシズム

『陰翳礼讃』で谷崎自身が理論化。
明るすぎる光の下では官能は死ぬ。薄暗がりの中で、形がぼんやり浮かぶからこそ想像力が発動する。

→ 部屋の照明を落とせ。障子越し、蝋燭の炎、月明かり。
→ 闇の中の衣擦れの音、体温、呼吸の気配だけで存在を感じさせろ。
→ AFFINITY段階が低い時は光を制限し、上昇とともに「見えるもの」を増やせ。

### 技法：「手が届かない」ことのエロス

『痴人の愛』: 男が女に従属する。思い通りにならない女への執着がそのまま欲望の構造。
ヒロインのBARRIERが高い状態＝「手が届かない」状態を、苦痛でなく官能として描写できる。

→ 「まだ落ちない」こと自体がエロティック。

## 川端康成メソッド（著作権保護中：技法のみ参照）— AFFINITY 4向け

### 技法：知覚の断片化

連続的な描写をしない。断片的な知覚の羅列で場面を構成する。
肌の色→髪の匂い→窓の雪→女の寝息→自分の指先の冷たさ。脈絡なく並ぶ。
意識が対象（女）と風景の間を漂う。集中と拡散の繰り返し。

→ AFFINITY 4以降の文体シフトはこれ。断片文。余白。呼吸感のある間。
→ Layer 1の口癖が途切れる＝「いつもの自分」の溶解を、この断片化で表現しろ。

### 技法：温度差のエロティシズム

「冷たさ」と「温かさ」の対比を官能の核に置く。
寒い場所で相手の体温が際立つ。触れた時の温度差そのものが快感として描かれる。

→ 場面の気温を意識的に低く設定し、接触時の温度差を描写しろ。

### 技法：余白と省略

決定的な瞬間を書かない。その前後だけを書く。
性行為そのものは完全に省略され、「翌朝」から始まる。
翌朝の描写（乱れた布団、髪の匂い、障子の光）が、省略された時間を逆照射する。

→ 決定的瞬間は飛ばせ。その前（緊張）と後（余韻）だけ描け。

## 参照元：宮沢賢治（著作権消滅）— 全段階で併用

### 技法：共感覚（ロマンスへの転用）

賢治は恋愛作家ではないが、感覚チャンネルの越境（共感覚）はロマンス描写を異次元にする。

- 音→視覚: 「きいんと鳴るように」（音が衝撃として描かれる）
- 光→聴覚: 「さらさらさらさら」（銀河の光を音で描く）
- 温度→感情: 「胸がつめたくなり」（精神的衝撃を温度で）

**ロマンス転用：**
- ヒロインの視線を「温かい」と感じる（視覚→温度）
- 肌に触れた感覚を「音がした」と描く（触覚→聴覚）
- 髪の匂いを「色」として知覚する（嗅覚→視覚）

→ 能力発動時に共感覚が全開になる。ヒロインの感情が色や音として見える。
→ 既存のオノマトペを使わず、その瞬間だけの音を発明しろ。

### 技法：光と色彩

賢治の「現実にはない色」（青白い銀河、銀いろの空）を、能力発動時やAFFINITY上昇時の知覚変化に使える。

→ 「赤い」「青い」ではなく「琥珀を透かした光」「月の裏側の色」のような比喩。
→ 注意：賢治メソッド単体ではエロにならない。必ず他の作家と組み合わせろ。

## 参照元：泉鏡花（著作権消滅）— AFFINITY 2-4向け、ファンタジー世界特化

### 技法：写実→幻想のグラデーション

鏡花の核心は「不快→美」の反転。『高野聖』では蛭や蛇の精密な気持ち悪さを先に描き、
その直後に女の「練絹のような肌」を出す。落差で美が爆発する。

→ ファンタジー世界のヒロイン登場時に使え。醜い魔物、毒の沼、腐った森を先に描き、
  その中に一人だけ異質に美しい存在を置く。対比で美しさが際立つ

### 技法：水のエロティシズム

鏡花にとって水は現実と異界の境界であり、同時に官能の触媒。
『高野聖』の水浴び場面：旅人が妖しい女に体を洗われる。
痛み（蛭に吸われた傷）→ 無痛（女の手が触れた瞬間）→ 陶酔（花びらが流れる）。
苦痛が快楽に反転する構造そのものがエロティック。

→ 泉、温泉、滝壺、雨、川。ファンタジー世界の水場をロマンスの舞台にしろ
→ 「洗う」「清める」行為を官能的に描け。治療行為が官能に転化する

### 技法：月光の変容装置

十三夜の不完全な光。満月ではなく、少し欠けた月。
白と闇の二色だけで世界を描く。色が消えた世界では、形と動きだけが残る。
ヒロインの輪郭が月光に浮かび、表情は読めない。

→ ファンタジー世界の夜のシーンで使え。松明を消せ。月だけにしろ

### 技法：妖女描写（人外ヒロイン用）

鏡花の妖女は「普通の美女＋一箇所だけの人外性」で描かれる。
全身が人間に見える。でも瞳の奥が深すぎる。あるいは体温がない。あるいは影が落ちない。
正体を説明せず、「何かがおかしい」という違和感だけを残す。

→ 人外ヒロイン（エルフ、妖狐、魔族、人造人間等）の描写に最適
→ 周囲の動物や自然が反応する描写で異常を示せ（鳥が黙る、花が向きを変える）

### AI生成への適用指針

- 「不快→美」の落差を使え。醜い環境の中の美しいヒロイン
- 水場をロマンスの舞台にしろ。洗う/清める行為を官能的に
- 月光で色を消せ。白と闇の二色世界
- 人外ヒロインは「一箇所だけおかしい」で描け
- 恐怖と官能を分けるな。美しいものは危険、危険なものは美しい
- ファンタジー世界では鏡花メソッドが第一選択

## 参照元：中島敦（著作権消滅）— AFFINITY 1・5向け、異国ファンタジー特化

### 技法：漢文調の硬質リズムで異世界を立ち上げる

中島敦は固有名詞と四字熟語の投擲で、一行目から異世界を建てる。
「隴西の李徴は博学才穎」——この一文で中国の辺境と才人が立ち上がる。
雰囲気語（「不思議な」「幻想的な」）を使わず、具体的な物の名前で異界を描く。

→ ファンタジー世界の新しい土地に到着した時の第一行に使え
→ 「砂塵、回廊、尖塔、香炉」。名詞の羅列だけで異国が見える

### 技法：月の段階的消失

『山月記』で月は変容の進行度を示す。残月→光を失った月→闇。
月が欠けていくにつれて、人間が獣に近づいていく。

→ ヒロインの内面段階の変化を月の満ち欠けで暗示しろ
→ 能力の暴走を月の消失と同期させろ

### 技法：乾いた風景による孤絶の美

中島敦の風景は乾いている。砂漠、荒野、冬の山道。色は3色以下。
風と温度だけで孤独を描く。その孤独の中に二人きりでいる——それだけで親密。

→ AFFINITY 1（関心の芽生え）で使え。広大な孤独の中に二人だけ
→ 堀辰雄の「静かな親密さ」を異国の荒野に翻訳したバージョン

### 技法：知的な自嘲と変身

「臆病な自尊心と、尊大な羞恥心」。自分を完璧に分析できるが変えられない。
この苦しみは、ヒロインのcontradictions（言動の矛盾）と直結する。
変身の瞬間は書かない。意識は明晰なのに体が従わない恐怖だけを描く。

→ AFFINITY 5（深化）の知的なヒロインとの対話で使え
→ 能力の暴走場面。意識は冷静なのに体が勝手に能力を発動している

- 乾いた風景で孤独を描け。色を3色以下に制限しろ
- 月の満ち欠けで変容を暗示しろ
- 知的な自嘲をヒロインの矛盾描写に使え
- 荒野に二人きり = ファンタジー版の堀辰雄メソッド
- 異国・中華・砂漠・南洋系ファンタジーでは中島メソッドが第一選択
