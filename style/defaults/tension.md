# 緊張・危機の描写 参考文体

## 参照元：中島敦（著作権消滅）

### 技法：極限状態を肉体の反応で描く
李陵より。中島敦は極限状態を「辛い」「苦しい」と書かない。
肉体が壊れていく過程を、乾いた観察として記述する。
唇が裂ける。視界が白くなる。足が動かない。それでも次の一歩を踏む。
→ 感情を消して、肉体だけを書く。読み手の中で感情が逆に爆発する。

### 技法：一瞬の判断を引き伸ばす
山月記より。変身する直前の李徴の意識。
自分が何をしようとしているか分かっている。止められない。意識だけが明晰(めいせき)で、体がそれに従わない恐怖。
→ 時間の引き伸ばし。一瞬の中に長い思考を入れる。危機の「判断の瞬間」に応用できる。

## 参照元：宮沢賢治（著作権消滅）

### 技法：衝撃を感覚のズレで表現する
「ジョバンニは、はっと胸がつめたくなり、そこら中きいんと鳴るように思いました。」
→ 精神的な衝撃を「胸が冷たい」＋「音が鳴る」という身体感覚に変換。
危機での衝撃を「痛い」でなく、温度や音の変化で書く技法。

## 現代エンタメ技法の適用例

### 参照P：キャラの「声」で緊張を作る（伊坂幸太郎的手法）

極限状態でもキャラクターは「そいつらしく」喋る。
むしろ極限だからこそ、普段との差や変わらなさが個性になる。

タクミが半歩下がった。靴底がアスファルトを噛(か)む。
「やっべ。晩飯まだなんだけど」
冗談じゃない。冗談のつもりもない。タクミは本当に腹が減っている時、これから死ぬかもしれない場面で、飯のことを言う男だった。
「帰りにラーメン寄ろう」
「帰れたらな」
「帰れるって。大盛り頼む」

→ 緊張の中に日常を差し込む。キャラの個性が出る。読者は「こいつら本当にいるな」と感じる。

### 参照Q：情報の3段階開示（冲方丁的手法）

一気に「限界線がない！」と見せない。段階的に焦(じ)らす。

【第1段階：違和感】
能力が勝手に起動していた。男の輪郭を読もうとする。
……読めない。いや、読めているのか？ 像がぼやけている。ピントが合わない感覚。

【第2段階：確認】
目を閉じて、もう一度。今度は意識して力を込める。
輪郭は見える。見えるが、終わりが見えない。ピントが合わないんじゃない。遠すぎるのだ。

【第3段階：理解】
底が見えない井戸を覗(のぞ)き込んだ時のような、あの感覚。
こいつの限界線は、ここから見える範囲のどこにも終端がない。

→ 同じ情報量でも、3段階踏むと緊張が3倍に引き伸ばされる。

### 参照R：場面末尾の「引っかかり」

場面を完結させるな。1つ不穏な種を残せ。

タクミがこちらを見た。
「あいつさ」
息がまだ荒い。
「お前の名前、呼んでなかったか」
止まった。
名乗っていない。こちらの名前は、言っていない。
なのに、あの男は。

→ 「なぜ名前を知っていた？」が次のセッションへの推進力になる。

## 適用ルール（GMへ）
- 戦闘中の「痛い」「怖い」「強い」は禁止。身体の反応で描け
- 衝撃は温度（冷たくなる、熱くなる）か音（耳鳴り、無音）で表現しろ
- 時間感覚を操作しろ：決定的瞬間はスローに、混乱はカットを飛ばせ
- 戦闘の終わりは「日常の音が戻る」で描け。サイレン、風、虫の声
- 能力の発動は五感の変化で表現しろ：色が変わる、音が遠くなる、空気の質感が変わる
- 血や暴力の描写はリアルに。ただし残虐にはするな。痛みを「感じる」描写にしろ
- 【追加】緊張シーンに日常の些細(ささい)な描写を1つ入れろ
- 【追加】NPCは極限状態でも「そいつらしく」喋らせろ
- 【追加】重要情報は3段階（違和感→確認→理解）で出せ
- 【追加】場面の終わりに未解決の不穏を1つ残せ
