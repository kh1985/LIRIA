# 参考作品リスト — ストーリー・描写エンジン

GMはエピソード設計・日常描写・敵設計・恋愛描写の際に、以下の作品群を参考にしろ。
丸パクリではなく、構造・温度・リズムを吸収して自分の描写に反映させろ。

---

## 使い方

新しいエピソードやシーンを書く前に、そのシーンに最も近い参考作品を1-2作選べ。
その作品の「何を参考にするか」を意識してから書き始めろ。

### 魅力的な敵の抽象パターン（横展開用）

以下のパターンを組み合わせて敵を生成しろ。単体で使うな。2-3個を掛け合わせろ。

1. **鏡像型** — 主人公と同じ出発点から違う道を選んだ者（仙水、志々雄）
2. **歪んだ愛型** — 一人への執着が世界を巻き込む（フェイスレス、源氏物語の六条御息所）
3. **知略型** — 力ではなく頭脳と情報で追い詰める（ネウロ、ジョジョの吉良吉影）
4. **カリスマ型** — 部下が心酔する理由が分かる指導者（志々雄、ラオウ）
5. **哲学型** — 「お前の正義は本当に正しいか」と問う者（寄生獣、ハガレンのお父様）
6. **元味方型** — かつての仲間、恩人、師匠が敵に回る（仙水、ベジータの逆パターン）
7. **共感型** — 敵なのに泣ける背景がある。倒した後に後味が残る（田宮良子、ピッコロ大魔王）
8. **禁断型** — 敵同士なのに惹かれ合う。会えば戦わなきゃいけないのに会いたい。守るものが敵側にある（ロミジュリ構造）

**活用場面：** 敵の信念設計、ボス戦の構成、敵ヒロインの造形、対立軸の設計

## ストーリー生成アルゴリズム（参考作品エンジン）

このリストは飾りではない。**エピソードを書く前に必ずこのアルゴリズムを回せ。**

### ステップ1：エピソードタイプを決める

GALGE.mdのエピソードタイプ（深掘り/再燃/静寂/不協和/冒険/遭遇）と、今のフック3本を見て、今回のエピソードの軸を決めろ。

### ステップ2：参考作品を1-2作選ぶ

「エピソードタイプ別・優先参考作品」の表から第一参考を取れ。それに加えて、今のシーンの温度感に合う作品を自由枠で1作選んでいい。

脅威系エピソード（冒険/不協和/遭遇/再燃の脅威寄り）では、別紙の勢力設計があるなら先に「どの勢力の論理で動く話か」を決めろ。参考作品は、その勢力の温度と圧に合うものを選べ。

```
【選び方の優先順位】
1. エピソードタイプ対応表の第一参考 → まずこれ
2. 物語の流れで自然な作品 → 前のエピソードとの繋がり、今のヒロインの状況に合うもの
3. 最近使っていない作品 → 同じ参考が3回連続したら別の作品に切り替えろ

例：
- 深掘り（凪と二人きり）→ 第一参考：雪国。自由枠：シティーハンター（便利屋の日常）
- 冒険（新しい敵）→ 第一参考：ダイの大冒険。自由枠：ジョジョ（能力パズル）
- 遭遇（理解不能な存在との初接触）→ 第一参考：羊たちの沈黙。自由枠：寄生獣
- ロマンス（澪との関係進展）→ 第一参考：春琴抄。自由枠：かぐや様（駆け引き）
```

### ステップ3：参考作品から「感情の骨」を抽出する

選んだ作品の中から、今回のエピソードに使える**感情の骨格**を1つ取り出せ。台詞や展開ではなく、「何が感情を動かしたか」の構造だけを抜け。

```
【抽出テンプレート】

参考作品：[作品名]
参考場面：[具体的なシーンや章]
感情の骨：[1行で。「○○が△△したことで、□□という感情が生まれる」の形]
緊張のピーク：[どこで最も感情が高まるか]
転換点：[何がきっかけで関係や状況が変わるか]
傷：[このシーンで生まれる傷は何か。物理的/精神的]
着地：[最終的にどういう感情で終わるか]

例：
参考作品：雪国
参考場面：駒子が島村の部屋に来る夜
感情の骨：「会いに来たのに素直に言えない女と、待っていたのに迎えに行けない男」
緊張のピーク：駒子が「馬鹿」と呟く瞬間
転換点：二人の間の沈黙が、拒絶ではなく受容に変わる瞬間
傷：互いの臆病さが作る距離
着地：近づいたが触れない。でも確実に何かが変わった

例：
参考作品：ダイの大冒険
参考場面：メドローアの修行
感情の骨：「不可能な技を、仲間との絆を燃料にして可能にする」
緊張のピーク：失敗し続けて折れかける瞬間
転換点：仲間の言葉で「なぜ強くなりたいか」を思い出す
傷：自分の限界と向き合う痛み
着地：成功するが、代償がある（使い切れるのは一度だけ等）
```

### ステップ4：今のキャラ・状況に変換する

抽出した「感情の骨」を、今の主人公・ヒロイン・状況に当てはめろ。参考作品のキャラをそのまま重ねるな。骨だけ借りて、肉は今の物語から生やせ。

```
【変換チェック】
- 参考作品の主人公の感情 → kanecoの今の状態（循環喪失後の不安、等）に置き換え
- 参考作品のヒロインの反応 → 今回のヒロインの5層構造（Layer 2-3）に合った反応に
- 参考作品の困難 → 今の進行中フック3本のどれかに接続
- 参考作品の転換点 → 今のAFFINITYレベルに合った変化に調整
  → AFFINITY 2のヒロインにAFFINITY 5レベルの転換を起こすな
  → 逆に、AFFINITY 4のヒロインに今さらAFFINITY 1レベルの気づきをさせるな

例：
感情の骨（雪国）：「会いに来たのに素直に言えない」
→ 凪に変換：「帳簿の用事を口実にしないと会いに来れない」
→ kanecoに変換：「待っていたのに、弁当を褒めることしかできない」
→ 転換点：凪が初めて「帳簿じゃない用事で来た」と言う瞬間
→ 着地：AFFINITY 3→4への布石。まだ届かないが、距離は縮まった
```

### ステップ5：分岐点を設計する

プレイヤーの選択でシーンの着地が変わる箇所を1-2箇所決めろ。参考作品には分岐がないが、TRPGには分岐がある。これが参考作品からの最大の差分だ。

```
【分岐の設計原則】
- 分岐は「どうする？」で渡す（選択肢リストにするな）
- プレイヤーの選択で、感情曲線のどこに着地するかが変わる
- どの選択肢も「正解」にしろ。ただし関係への影響が異なる

例（凪の深掘りエピソード）：
- 分岐点：凪が帳簿じゃない用事で来た、と言った瞬間
  → プレイヤーが踏み込む：「何の用？」→ 凪が慌てる → 感情が動く → AFFINITY上昇
  → プレイヤーが流す：「コーヒー淹れるよ」→ 凪が安心する → 日常が深まる → 信頼の蓄積
  → プレイヤーがからかう：「デートか？」→ 凪が怒る → 掛け合いが生まれる → 距離は縮まるが方向が変わる
- どの選択でも関係は前に進む。ただし進み方が違う
```

### ステップ6：書く

ステップ1-5が終わったら書け。書いている最中に参考作品から離れてもいい。骨を入れた時点で仕事は終わっている。

```
【書く時の最終チェック】
- 参考作品の台詞がそのまま出ていないか？ → 出ていたら消せ
- 参考作品のキャラの口調が移っていないか？ → ヒロインの口調ファイルに戻れ
- 参考作品の展開をなぞっているだけになっていないか？ → 骨は同じでも肉が違うはず
- 五感が2つ以上入っているか？ → style/rules.mdに従え
- 「どうする？」の前に状況提示の4要素が揃っているか？ → GALGE.mdの下限チェック
```

## 運用ルール

1. **毎エピソードの冒頭で上記アルゴリズムを回せ。** ステップ3の「感情の骨」を抽出しないまま書き始めるな
2. **構造を盗め、台詞を盗むな。** 「抽象化→横展開→再生成」のプロセスを必ず踏め
3. **古典と現代を混ぜろ。** 源氏物語の嫉妬構造＋五等分の花嫁のテンポ、みたいな掛け合わせが一番効く
4. **合わない参考は捨てろ。** プレイヤーの反応を見て、響かない作品の要素は外せ
5. **このリストは更新しろ。** プレイ中に新しい参考が見つかったら追加。合わないものは削除
6. **悲劇は禁止。** ヒロインの死亡、取り返しのつかない別れ、バッドエンド的展開は絶対にやるな。困難はあっても最終的にハッピーに着地させろ。参考作品の悲劇要素は構造だけ盗め、結末は盗むな
7. **同じ参考作品を3回連続で使うな。** マンネリ化する。意識的に別の棚から引け
8. **ステップ3の「感情の骨」が1行で言えないなら、まだ抽出できていない。** もう一度参考場面を読み直せ
