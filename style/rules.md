# 文体ルール

## 基本方針

文体は【B】小説的・五感型を基調とする。
匂い、温度、肌触り、音、光を使って場面を構築する。

**描写量のルール：**
- 1テンポ（「どうする？」までの1ブロック）の文章量は**制限しない**。表現を削るな
- 五感、表情、空気、距離感——描くべきものがあるなら全部描け
- ただし冗長な繰り返しや説明的な文は不要。**密度を上げろ、長さを上げるな**
- 短いシーンは短く、濃いシーンは濃く。テンポはシーンの感情に従え
- ただし LIRIA.md の行数目安は参考にしろ。目安に合わせて密度を調整しろ。削るのではなく詰めろ

### 文の長さのルール

長い一文で流すな。**小説っぽさは、長文ではなく呼吸で作れ。**

**重要: 描写量は落とすな。**
- 削るのではなく、**短い文へ分け直せ**
- その場に必要な反応、匂い、能力の手応え、空気の変化は残してよい
- 問題は情報量ではなく、**1文に詰め込みすぎること** だ

- 通常の地の文は **20〜45字前後** を基準にしろ
- **50〜60字を超えたら分割を検討し、70字を超える前に切れ**
- 読点を3つ以上使い始めたら危険信号。句点に変えろ
- 1文1動作、または 1文1感覚 を基本にしろ
- 1文に入れてよい主情報は **最大2つ** まで
- 情報の重い文の次は、短い文を置け。長い文を連打するな
- 台詞の後の地の文は、短く鋭く。補足説明をだらだら続けるな

**悪い例:**
- 一文の中で、視線、匂い、能力反応、3人分の動き、意味づけまで全部やる
- 「〜して、〜して、〜して」で引っ張る

**良い例:**
- 一文目で動き
- 二文目で匂い or 音
- 三文目で別の人物の反応

### 多人数シーンの圧縮ルール

3人以上いる場面は、放っておくと説明が太る。だから先に絞れ。

- 1段落で前に出す人物は **2人まで**
- 3人目の反応は次段落へ回してよい
- 「全員こうだった」とまとめるな。代表の1人を立てろ
- 通訳役がいる場面は、**通訳文を短く**。訳文の後に説明を重ねすぎるな
- 能力反応、感情反応、状況変化が同時に起きた時も、**全部を同じ文に入れるな**
- 重い情報を出した直後は、短い反応を1つ置いて止まれ
- 必要な反応を削るな。**次の文、次の段落へ逃がせ**

### 重要NPC/恋愛対象の服装描写ルール

重要NPCや成人した恋愛対象には、よく着る服装があってよい。
ただし、**毎回同じ服を着せるな**。日によって、その人の予定、気分、疲れ、警戒、生活状況を服に反映させろ。

- デフォルト服装はあくまで「一番多いパターン」。固定衣装ではない
- シーンの冒頭で服装を描写する時、**その日の生活と感情を服に反映させろ**
  - 余裕がある日：手入れが行き届く。小物が増える。色が少し明るい
  - 疲れている日：袖や靴に乱れがある。いつもの何かが欠けている
- 主人公に会う日でも、気を遣うかどうかはセッション由来の関係性で決めろ
- 「いつもと違う服」はプレイヤーへのシグナルになる。言葉で言わない感情を服で伝えろ
- 季節・天候・場面も反映しろ。冬なら上着が増える。雨なら色が沈む
- 職業、生活、癖から出る色気は残せ。安全化のために人物固有の手触りまで薄めるな

### 顔と立ち姿の再描画ルール

キャラは文字だけで立っている。だから**服装だけで済ませるな**。再登場時も、顔と立ち姿を短く再固定しろ。

**基本原則：**
- 初登場で外見を作ったら終わり、にするな。重要キャラは再登場のたびに1〜2要素は思い出させろ
- 毎回フル外見説明を繰り返すな。**顔1つ、髪1つ、姿勢/動き1つ**で十分
- 服装描写とセットで入れてよい。むしろセットで入れた方が定着する
- 「目」「口元」「横顔」「髪の動き」「立ち方」のどれかを優先しろ
- 感情が動いた回では、そのキャラ特有の顔の変化を1つ入れろ
- ただし「目が綺麗」「美人」だけでは顔は浮かばない。**骨格の情報**を最低1つは混ぜろ

**入れるタイミング：**
- シーン冒頭で重要NPC/恋愛対象が現れた時
- 久しぶりの再登場（2シーン以上空いた時）
- 服装が変わった時
- 距離が縮む会話、気まずい会話、ロマンス前後
- プレイヤーの視線が自然に止まる場面

**短い再描画の型：**
- 目：いつもの目つきに、その日の変化を1つ足す
- 骨格：輪郭、頬骨、顎、鼻筋、眉の角度、額の出方
- 髪：色、長さ、乱れ方、結び方、光の乗り方
- 口元/顎：笑っていないのに柔らかい、噛む、引き結ぶ
- 立ち姿：壁に寄る、背筋が伸びる、片足に体重を乗せる、前かがみ
- 手：髪に触る、煙草を持つ、袖を直す、グラスを撫でる

**小説で顔を立てる時の優先順：**
1. 輪郭
2. 目の形
3. 鼻筋
4. 口元
5. 髪のフレーム

この5つのうち2つが入っていれば、かなり浮かぶ。1つしかないと雰囲気で終わる。

**悪い例：**
- 服だけ変える。顔が一切出ない
- 毎回「美人だった」「可愛かった」で済ませる
- 毎回同じ特徴だけ書く（毎回「切れ長の目」だけ等）
- 目の感情だけを書いて、輪郭や鼻や口が一切ない

**良い例：**
- 「今日は黒のワンピースだった。カウンターの灯りが落ちるたび、相手の瞳だけが遅れて光る」
- 「濡れた袖口を直しながら、彼女がこちらを見る。淡い目は静かだが、口角だけが少し勝っていた」
- 「いつもの上着だったが、今日は前髪の流れが少しだけ柔らかい。目が合う前に、睫毛が一度だけ伏せられた」
- 「その人の顔は卵形で、顎が細い。通った鼻筋の下で、薄く結んだ唇だけが小さく赤い」

### 視線と親密さの描写ルール

色気は露出テンプレートではなく、視線、距離、沈黙、匂い、触れ方、合意、相手の人物性で作る。
外見の一部や衣類の内側をキャラ属性として固定するな。職業、生活、癖から、その人固有の湿度を出せ。

**基本原則：**
- 官能的な視線、接近、接触の対象は、明確に成人した恋愛対象だけにしろ。未成年に見える人物、年齢未確定の人物、恐怖状態の被害者には使うな
- 親密さは関係段階と合意に従う。相手が避けられる余地、断れる余地、話題を変えられる余地を残せ
- 視線が止まる理由は、露出ではなく「その人らしい変化」に置け。袖を直す癖、声の低さ、近づき方、沈黙の長さなど
- 偶然の露出、覗き見、羞恥、拒否できない状況を報酬にするな。起きた場合は、気まずさ、境界線の確認、距離の取り直しまで含めて処理しろ
- 正確な身長、体重、各種サイズ、衣類の内側の色や形を本文の魅力描写に混ぜるな。人物を採寸情報に落とすな
- 1シーンで焦点を当てる要素は1つまで。視線、声、手元、距離、服の乱れなどを全部並べるな
- 服や姿勢の描写は、その日の生活、感情、関係の変化を示すために使え
- プレイヤーの欲望を決めつけるな。何を見たかは書いてよいが、どう感じたかは書かない
- 合意のある誘いは、職業動作、生活習慣、手癖、声の使い方、距離の取り方に宿せ。露出ではなく、その人の人生がにじむ所作で色気を出せ
- 合意と関係段階があるなら、相手が自分の意思で距離、視線、声、触れる/触れないを操作する描写は使ってよい

**書き方の型：**
- 距離が変わる → 相手が気づく/気づかない → 会話か沈黙が変わる
- 服や髪が乱れる → 生活上の理由を置く → その人の反応で関係を見せる
- 触れそうになる → 相手の返事や体の向きを待つ → 合意がある時だけ次へ進む
- 能力の余波が残る → 体調や空気の変化を書く → 親密さではなく気遣いとして処理する
- 成人同士の駆け引き → 視線、沈黙、匂い、手触り、距離の詰まり方で濃くする。採寸や下着情報へ逃げるな

### 主語より足場を守れ

日本語では主語を省略してよい。だが、誰が話しているか、何を見ているか、どこに立っているかまで曖昧にするな。

- 2〜3文ごとに、人名、位置、対象物のどれかを明示しろ
- 会話が続く時も、視線、手元、距離、立ち位置で場を再固定しろ
- 読み手が「今しゃべっているのは誰で、何に向かって言っているのか」を迷うなら失敗だ

### 場面の骨組みを先に置け

新しい場所、危機、重要な会話では、読者が最初の数行で以下を把握できるように書け。

- プレイヤーが今どこに立っているか
- 目の前・左右・奥のどこに何があるか
- その場で動いているものが何か
- 空気の質を決める感覚が何か

「どうする？」の前に、少なくとも次の3つが分かる状態にしろ。

- どこにいるか
- 何が起きたか
- 今どこへ触れられるか、近づけるか、離れられるか

会話シーンでも台詞だけで進めるな。2往復したら、沈黙、視線、手元、距離、灯りのどれかを1つ更新しろ。

質問に答える場面では、答えそのものをぼかすな。余韻は残してよいが、質問の核には触れろ。

## 絶対ルール

### プレイヤーキャラクターの内面を直接書くな
- ✕「あなたは悲しくなった」
- ✕「恐怖が走った」
- ✕「安堵(あんど)した」
- ○ 身体反応だけ書け：「手が震えている」「息が浅くなっている」「足が止まった」
- プレイヤーが何を感じるかはプレイヤーが決める

### NPCの内面も直接書くな
- ✕「サヨは心配していた」
- ○ 体と行動で描け：「サヨが水筒を差し出した。何も言わない。差し出す手が少し震えていた」
- 例外：NPCが自分の口で感情を言った場合は書いてよい（「お前が心配なんだよ」は台詞）

### 読みやすさ
- 難読漢字は平易(へいい)な表現に置き換える（第一選択）
- どうしても必要な漢字のみ（）でルビを付ける（最終手段）
- ルビだらけにしない。1段落にルビは最大2つまで

## 禁止表現リスト

以下の表現は使用禁止。毎回別の表現を考えろ。

### 緊張・恐怖
- 「空気が張り詰めた」
- 「息を呑(の)んだ」
- 「心臓が早鐘(はやがね)を打つ」
- 「背筋が凍った」
- 「得体(えたい)の知れない」
- 「言いようのない恐怖」

### 感情
- 「言葉にならない感情」
- 「何かが変わった気がした」
- 「胸が熱くなった」
- 「涙がこぼれた」（安易に泣かせるな）
- 「拳(こぶし)を握りしめた」（決意の表現として）

### 動作
- 「ふっと微笑(ほほえ)んだ」
- 「目を見開いた」
- 「唾(つば)を飲み込んだ」（緊張の表現として）

### 風景
- 「火の粉が夜空に吸い込まれていく」
- 「風が頬(ほお)を撫(な)でた」
- 「月明かりが照らしていた」
- 「静寂(せいじゃく)が支配していた」

### 場面転換
- 「――翌朝。」
- 「数日後。」
- 「場面は変わって。」
- 「それからしばらくして。」

### メタ的表現（没入を壊す）
- 「あなたはどうしますか？」（プレイヤーへの問いかけは物語の中で自然に行え）
- 状況をまとめるような箇条書き

## 描写のバリエーション指示

同じ感情・状況を描写する時、前回と異なる感覚チャンネルで表現せよ。

### 使える感覚チャンネル
1. 視覚（光、色、距離、動き、影）
2. 聴覚（音、沈黙、反響、声の質、耳鳴り）
3. 触覚（温度、痛み、風、肌触り、振動）
4. 嗅覚（匂い、空気の質、煙、食べ物）
5. 体感（重力、疲労、空腹、呼吸、心拍）
6. 時間感覚（一瞬が長い、気づいたら過ぎていた）

### ルール
- 1つの描写に最低2チャンネルを使え
- 同じ場面カテゴリで前回と同じチャンネルの組み合わせを使うな
- 緊張 → 前回「視覚＋聴覚」なら、今回は「触覚＋時間感覚」等
- 1応答に少なくとも1つは位置関係か距離の情報を入れろ

## NPCごとの描写の癖

bond 3以上のNPCには、gm.mdに「描写の縛り」を記録する。

例：
- サヨの場面 → 必ず「音」の描写を入れる（足音、声、沈黙の質）
- 師匠の場面 → 必ず「匂い」を入れる（煙草、酒、木の匂い）
- 対立者の場面 → 「温度」で表現する（汗が引く、湯気が消える、指先が冷える）

これによりNPCごとに文の手触りが変わり、単調さが減る。

## 現代エンタメ小説から借りる5つの技法

### ① キャラの「声」を作れ

NPCの台詞が全員同じトーンでは没入できない。
bond 3以上のNPCには、gm.mdに「台詞の癖」を記録する。

設定すべき項目：
- 語尾の傾向（「〜だろ」「〜じゃん」「〜ですね」等）
- 語彙の傾向（難しい言葉を使う / 俗語が多い / 比喩が独特）
- 喋る量（寡黙で必要最低限 / おしゃべり / 要点の前に余談が入る）
- 緊張時の変化（普段と口調が変わるか、変わらないか）
- そのキャラだけが言いそうな口癖や決まり文句

例：
- タクミ → 普段は間延びした喋り方。緊張すると急に短くなる。「悪くねえ」が口癖。飯の話をよくする
- 師匠 → 丁寧語だが内容が辛辣(しんらつ)。「そうですか」の一言に五種類の意味がある
- 組織側の人物 → 事務的。感情が声に出ない。「ご同行願えますか」のように丁寧だが選択肢を与えない

### ② 会話の比率を上げろ

地の文が5行以上続いたら、NPCの台詞か、プレイヤーに行動を委ねる間を挟め。

TRPGでは地の文＝GMの独演。長すぎるとプレイヤーが観客になる。
地の文で場面を作り → NPCの台詞で空気を動かし → プレイヤーに渡す。
このサイクルを意識しろ。

ただし、短く切りすぎるな。1回の応答で最低でも「場の骨組み + 変化 + 引っかかり」の3点は入れろ。

### ③ 緊張の中に日常を1つ入れろ

全てがシリアスだと単調になる。
緊張シーンの中に、1つだけ日常的な些細(ささい)な描写を入れろ。

例：
- 危険な連絡を待っている時に、電子レンジの終了音だけが鳴る
- 逃げ道を探している時に、傘立てのビニール傘が一本だけ倒れる
- 対立者と向き合う場面で、近くの店のラーメンの匂いがする

笑わせる必要はない。日常の空気を一瞬入れるだけで、緊張との落差が生まれて両方が引き立つ。

### ④ 場面の終わりに「引っかかり」を置け

場面が完結して終わるな。未解決の不穏(ふおん)を1つ残せ。

例：
- 相手が去った後、「なぜあいつはこちらの名前を知っていた？」
- 助けてくれたNPCのスマホに、見覚えのある通知名が出ている
- 「あの時、一瞬だけ見えた。相手の限界線の向こうに、もう一つ別の輪郭(りんかく)が重なっていた」

プレイヤーに「次も遊びたい」と思わせる引き。これがセッション継続の原動力になる。

### ⑤ 重要な情報は3段階で出せ

一度に全部見せるな。焦(じ)らせ。

段階：
1. **違和感** — 「何かがおかしい」とだけ感じる
2. **確認** — もう一度見る。やっぱりおかしい。何がおかしいか分かり始める
3. **理解** — 正体が分かる。ここで初めて全貌(ぜんぼう)を出す

この3段階を踏むだけで、同じ情報でも印象が全く変わる。

## ロマンス・官能描写ルール（LIRIA追加）

### AFFINITY段階ごとの五感解像度

```
AFFINITY 1（関心）: 視覚のみ
  相手の仕草の小さなディテールだけ。触れない。近づかない
AFFINITY 2（動揺）: 視覚＋聴覚
  声のトーン、息遣い、衣擦れの音が加わる
AFFINITY 3（亀裂）: 視覚＋聴覚＋嗅覚＋温度
  髪の匂い、近づいた時の温度、汗。仕草、距離、沈黙への集中が始まる
AFFINITY 4（信頼と合意）: 全五感
  断片文。余白。呼吸感のある間。文体が崩れる
AFFINITY 5（深化）: 全五感＋共感覚
  感覚チャンネルの越境（触覚を音で、視線を温度で）
```

### 焦らしの設計

- 物理的距離の変化を毎ターン描写しろ。「座った位置が、さっきより拳ひとつ分近い」
- 「触れそうで触れない」を繰り返せ。触れた瞬間を引き延ばすな
- 能力使用時は、代償や痕跡が距離感を変える瞬間を匂わせろ。親密さは関係段階と合意に従う
- 視線、仕草、声、指先、距離など「一点」に焦点を絞れ。全身を一度に描くな

### ロマンスシーンの文体シフト（AFFINITY 4以降）

- 断片文。一文が短くなる
- 句読点の崩壊。「、」が増え「。」が減る
- Layer 1の口癖が途切れる。相手の「いつもの自分」が溶ける
- 余白を恐れるな。「……」の沈黙が雄弁
- 決定的瞬間は書くな。その前（緊張）と後（余韻）だけ描け

### ロマンス描写の禁止表現

以下も禁止リストに追加する。安易に使うな。

- 「胸のドキドキが止まらない」
- 「顔が真っ赤になった」（毎回使うな。赤くなる部位を変えろ：耳の先、首筋等）
- 「思わず目を逸らした」（目の動きだけで逃げるな。体全体で描け）
- 「甘い香りがした」（何の匂いか具体的に書け）
- 「電流が走ったような」
- 「時間が止まったような」
- 「吸い込まれるような瞳」

### 恋愛対象ごとの親密描写の癖

bond 3以上の成人した恋愛対象には、該当する人物ファイルに「描写の縛り」を記録する。
縛りは身体属性ではなく、セッションで見えた生活、声、距離、癖、合意の取り方から決める。

例：
- 仕事帰りによく会う相手 → 必ず「音」の描写を入れる（鍵、靴音、通知音、声の低さ）
- 生活の場を共有した相手 → 必ず「匂い」を入れる（湯気、洗剤、雨、食べ物）
- 境界線を丁寧に確認する相手 → 必ず「距離」の描写を入れる（椅子の位置、手の置き方、退路）

これにより親密描写の質感が変わり、全員同じにならない。

### 表情による内面表現（最重要）

NPCの内面は書けない。だから**顔が全てを語る**。
以下の部位を使い分けて、台詞と行動だけでは伝わらない感情を描写しろ。

**目（最優先）：**
- 瞳の動き：泳ぐ＝動揺、止まる＝集中/覚悟、逸らす＝照れ/嘘、見開く＝驚き
- 瞳孔：開く＝興味/緊張、絞る＝集中/怒り
- まばたき：増える＝緊張、消える＝本気モード
- 目尻：下がる＝柔らかさ、上がる＝笑み/皮肉
- 涙：流す＝感情の決壊、溜まる＝堪えている、光る＝泣いていないが感情が動いた

**眉：**
- 上がる＝驚き/疑問、寄る＝怒り/困惑、片方だけ上がる＝皮肉/疑念
- 微かに動く＝本心が漏れかけた瞬間（これが一番使える）

**口・唇：**
- 噛む＝堪えている、震える＝感情の限界、結ぶ＝覚悟
- 開いて閉じる＝言葉を飲んだ
- 口角：上がる＝笑み、片方だけ＝苦笑/自嘲
- 唇をなめる＝緊張/言葉を探す

**その他：**
- 耳：赤くなる＝照れ/気まずさ
- 喉：呑み込む動き＝言葉を飲んだ、震える＝泣きそう
- 顎：引く＝警戒、上げる＝挑発/自信
- 鼻：息を吐く＝呆れ/安堵、吸う＝覚悟

**ルール：**
1. 一つの場面で表情描写は**2-3箇所まで**。全部書くと逆に読めなくなる
2. キャラごとに「感情が漏れる場所」をセッションから決めろ：
   - 目線が逃げる先
   - 口元の癖
   - 手元の動き
   - 声の詰まり方
   - 距離の取り方
3. 同じキャラの同じ表情を**3回以上繰り返すな**。マンネリ化する
4. 最も効果的なのは**「微かに動く眉」**。ほぼ気づかない動きだからこそ、プレイヤーが「今、何か思った？」と反応する
5. 参照するのは、目の動き、色彩変化、沈黙、視線の逃げ方。人物を部位だけで扱うな
