# 起動方法

## 新規ゲーム

```bash
bash play.sh new
```

## 名前付きで新規 session 作成

```bash
bash play.sh new liria session_002
```

## 続きから再開、最新 session を自動選択

```bash
bash play.sh resume
```

## session 指定で再開

```bash
bash play.sh liria resume session_001
```

## Codex で再開

```bash
bash play.sh resume -codex
```

## session 一覧

```bash
bash play.sh list-sessions
```
